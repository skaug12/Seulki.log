# HFK 데일리 에이전트

오늘의 작업 기록과 미리알림 관리를 통합 실행합니다.

## 사용법

```
/run-daily
```

## 실행 워크플로우

### Step 1: 작업 일지 (daily)

1. 오늘 날짜의 daily 파일 확인/생성
   - 경로: `daily/YYYY-MM-DD.md`
2. 기존 내용이 있으면 보여줍니다
3. 추가할 작업 내용을 입력받아 기록합니다

### Step 2: 미리알림 관리

1. MCP Apple Reminders `list`로 "HFK" 리스트의 현재 미리알림을 조회합니다
2. 완료된 항목과 남은 항목을 보여줍니다
3. 새로운 To-Do 추가 여부를 확인합니다:
   - "예" → 내용과 마감일을 입력받아 MCP `create` / `bulk_create`로 추가
   - "아니오" → 종료

### Step 3: 오늘의 요약

- 기록된 작업 수
- 현재 미리알림 현황 (완료/미완료)
- 새로 추가된 To-Do 수
