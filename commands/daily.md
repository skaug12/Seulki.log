# HFK 데일리 작업 일지

오늘의 작업을 기록하고 업데이트합니다.

## 사용법

```
/daily
```

## 실행

1. 오늘 날짜로 daily 파일 경로를 확인합니다:
   - 경로: `daily/YYYY-MM-DD.md`
   - 예: `daily/2026-01-30.md`

2. 파일이 없으면 새로 생성합니다:
   ```markdown
   # YYYY-MM-DD 작업 일지

   ## 오늘 완료한 작업

   (작업 내용을 여기에 추가)

   ---

   ## 메모

   ```

3. 파일이 있으면 내용을 읽어서 보여줍니다.

4. 사용자에게 질문합니다:
   - "추가할 작업이 있나요?"
   - 있으면: 작업 내용을 입력받아 `## 오늘 완료한 작업` 섹션에 추가
   - 없으면: 현재 상태 유지

5. 작업 추가 형식:
   ```markdown
   ### N. 작업 제목

   작업 설명

   **수정된 파일**:
   - `파일경로` - 변경 내용

   ---
   ```

6. 작업 추가가 끝나면 사용자에게 질문합니다:
   - "미리알림에 추가할 To-Do가 있나요?"
   - 있으면: To-Do 항목을 입력받아 아이폰 미리알림에 추가
   - 없으면: 종료

7. MCP Apple Reminders 도구로 미리알림을 추가합니다:
   - `create` 또는 `bulk_create` 도구 사용
   - 리스트: "HFK"
   - 마감일: 기본 다음 날 오전 9시 (ISO 8601 형식)

---

## 작업 번호 자동 증가

기존 작업이 있으면 번호를 자동으로 증가시킵니다:
- 기존: ### 1., ### 2.
- 새 작업: ### 3.

---

## 예시

**입력**: "상품 템플릿 수정"

**결과**:
```markdown
### 3. 상품 템플릿 수정

(사용자가 입력한 상세 내용)

**수정된 파일**:
- `templates/상품 템플릿.md` - (변경 내용)

---
```
