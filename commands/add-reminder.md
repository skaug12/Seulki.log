# HFK 미리알림 추가

아이폰 미리알림(Reminders)에 To-Do 항목을 추가합니다.

## 사용법

```
/add-reminder
```

## 실행

1. 사용자에게 질문합니다:
   - "추가할 미리알림 내용을 입력해주세요"
   - 여러 개인 경우 줄바꿈으로 구분

2. 마감일을 확인합니다:
   - "마감일을 지정하시겠습니까? (기본: 내일 오전 9시)"
   - 옵션: 오늘, 내일, 이번 주 금요일, 다음 주 월요일, 직접 입력

3. MCP Apple Reminders 도구를 사용하여 미리알림을 추가합니다:
   - 1개인 경우: `create` 도구 사용
   - 여러 개인 경우: `bulk_create` 도구 사용
   - 리스트: "HFK"
   - 마감일 형식: ISO 8601 (예: "2026-02-13T09:00:00")

4. 추가 완료 메시지를 표시합니다:
   - "N개의 미리알림이 추가되었습니다"
   - 추가된 항목 목록 표시

---

## 날짜 설정

- 오늘: 현재 날짜 18:00
- 내일 (기본): 다음 날 09:00
- 이번 주 금요일: 해당 금요일 09:00
- 다음 주 월요일: 해당 월요일 09:00

---

## 예시

**입력**:
```
상품 사진 촬영
시즌레터 작성
재고 확인
```

**실행 결과**:
```
✓ 3개의 미리알림이 HFK 리스트에 추가되었습니다
  - 상품 사진 촬영 (내일 09:00)
  - 시즌레터 작성 (내일 09:00)
  - 재고 확인 (내일 09:00)
```
