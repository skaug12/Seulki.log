# Markdown 후기 → HTML 변환

마크다운으로 작성된 후기를 순수 HTML 코드로 변환합니다. CSS나 inline style 없이 시맨틱 HTML 태그만 사용합니다.

## 사용법

```
/convert-review-to-html <파일경로>
```

예시:
```
/convert-review-to-html note/리뷰/2026 봄시즌 북토크 후기.md
```

## 실행 절차

### 1단계: 파일 읽기

$ARGUMENTS에서 파일 경로를 추출하여 파일을 읽습니다.

파일 구조:
```
---
category: 리뷰
date: ''
idx: ''
images: []
status: draft
title: '제목'
---

# 제목

> **카테고리**: 리뷰
> **작성일**: ...

---

{본문 콘텐츠}
```

1. YAML frontmatter (`---` ~ `---`) 파싱
2. Obsidian 메타 블록 분리 (H1 제목, blockquote 메타, `---` 구분선)
3. 본문 콘텐츠만 추출 (메타 블록 이후)

### 2단계: 마크다운 → HTML 변환

아래 규칙으로 변환합니다. **CSS, style 속성 일절 사용하지 않습니다.**

| 마크다운 | HTML |
|---------|------|
| `## 소제목` | `<h2>{소제목}</h2>` |
| `### 소제목` | `<h3>{소제목}</h3>` |
| `**굵게**` | `<strong>{텍스트}</strong>` |
| `*기울임*` | `<em>{텍스트}</em>` |
| `[링크](URL)` | `<a href="URL">{링크텍스트}</a>` |
| `![alt](이미지URL)` | `<img src="이미지URL" alt="alt">` |
| `- 리스트` | `<ul><li>{항목}</li></ul>` |
| `1. 순서` | `<ol><li>{항목}</li></ol>` |
| `> 인용` | `<blockquote><p>{텍스트}</p></blockquote>` |
| `---` | `<hr>` |
| 빈 줄 | (무시 — HTML 구조로 충분) |
| 일반 텍스트 | `<p>{텍스트}</p>` |

**변환 규칙:**
- 연속된 리스트 항목(`-` 또는 `1.`)은 하나의 `<ul>` 또는 `<ol>`로 묶기
- 연속된 `>` 인용은 하나의 `<blockquote>`로 묶기
- inline 마크다운(`**`, `*`, `[]()`등)은 `<p>` 안에서 변환
- `![](IMAGE_PLACEHOLDER)`도 그대로 `<img src="IMAGE_PLACEHOLDER">`로 변환
- `<br>` 태그 불필요 — 블록 요소로 충분

### 3단계: 결과 출력

변환된 HTML 코드를 코드블록으로 출력합니다.

```html
<h2>소제목</h2>
<p>본문 텍스트</p>
<img src="IMAGE_PLACEHOLDER">
...
```

사용자가 복사해서 바로 사용할 수 있도록 **본문 HTML만** 출력합니다 (frontmatter, 메타 블록 제외).

## 참고

- **금지 단어**: "단순한", "넘어"(뛰어넘어/넘어서 등 포함), "리듬", "실질", "여정" — 변환 전 원문에 이 단어가 있으면 다른 표현으로 수정 후 변환
- style, class 속성을 절대 추가하지 않음
- `<html>`, `<head>`, `<body>` 래퍼 불필요 — 본문 조각(fragment)만 출력
- H1(`#`)은 메타 블록 제목이므로 본문 변환에서 제외
