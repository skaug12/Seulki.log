# 새 컴퓨터 셋업 가이드

## 빠른 셋업

```bash
# 1. Repo clone
git clone https://github.com/skaug12/Seulki.log.git ~/Seulki.log

# 2. 셋업 스크립트 실행
cd ~/Seulki.log && bash setup.sh

# 3. .env 파일 복사 (수동)
scp <기존컴퓨터>:~/Seulki.log/.env ~/Seulki.log/.env
```

## 워크스페이스 구조

```
~/
├── Seulki.log/              ← git: 스킬 + 스크립트
│   ├── .claude/commands/    ← Claude Code 스킬 (32개)
│   ├── scripts/             ← Python 스크립트
│   ├── data/                ← 운영 데이터
│   ├── docs/                ← 문서
│   └── .env                 ← API 키 (수동 복사)
│
├── hfk-imweb/  → (symlink)  ← iCloud: Obsidian vault
│   ├── product/             ← 상품 노트 (시즌별)
│   ├── note/                ← HFK NOTE
│   ├── images/              ← 이미지
│   ├── mcp-servers/         ← MCP 서버 코드
│   ├── daily/               ← 데일리 작업 일지
│   └── .google_credentials.json
│
└── .claude/
    └── commands/ → (symlink) ← ~/Seulki.log/.claude/commands
```

## 동기화 방식

| 분류 | 경로 | 동기화 | 비고 |
|------|------|--------|------|
| 스킬/스크립트 | `~/Seulki.log/` | git | `git pull`로 최신화 |
| 콘텐츠/이미지 | `~/hfk-imweb/` | iCloud (Obsidian) | 자동 동기화 |
| MCP 서버 | `~/hfk-imweb/mcp-servers/` | iCloud | vault와 함께 동기화 |
| 인증 파일 | `~/hfk-imweb/.google_*` | iCloud | vault와 함께 동기화 |
| API 키 (.env) | `~/Seulki.log/.env` | 수동 복사 | secrets |
| Python venv | `~/Seulki.log/.venv/` | 재생성 | `setup.sh`가 처리 |

## 상세 설정

### 1. .env 파일

`~/Seulki.log/.env`에 포함되어야 할 키:

- `GOOGLE_API_KEY`
- `HFK_CALENDAR_ID`
- `GOOGLE_CLIENT_ID` / `GOOGLE_CLIENT_SECRET`
- `GA4_PROPERTY_ID`
- `SLACK_BOT_TOKEN`

### 2. Obsidian

Obsidian → Open folder as vault → `~/hfk-imweb` 선택
(iCloud 동기화가 완료된 후)

### 3. 외부 프로젝트 (필요시)

```bash
# 핸드아웃 웹앱
git clone <repo> ~/Projects/session-guide

# 주문 동기화
git clone <repo> ~/imweb-order-sync
```

### 4. 경로 규칙

commands 내 모든 경로는 `$HOME/` 기반:
- `$HOME/hfk-imweb/` — Obsidian vault (symlink)
- `$HOME/Seulki.log/` — 스킬/스크립트
- `$HOME/google_sheet_search/credentials.json` — Google 인증
- `$HOME/Downloads/` — 다운로드

`/Users/sklee/` 같은 절대 경로는 사용하지 않습니다.
