# 새 컴퓨터 셋업 가이드

## 1. git clone

```bash
git clone <repo-url> ~/Seulki.log
```

## 2. .env 파일 복사 (git에 포함 안 됨)

`.env` 파일 위치: `~/Seulki.log/.env` (프로젝트 루트)

기존 컴퓨터의 `~/Seulki.log/.env`를 새 컴퓨터의 같은 위치에 복사합니다:

```bash
scp ~/Seulki.log/.env {새컴퓨터}:~/Seulki.log/.env
```

포함되어야 할 키 목록:
- `GOOGLE_API_KEY`
- `HFK_CALENDAR_ID`
- `GOOGLE_CLIENT_ID` / `GOOGLE_CLIENT_SECRET`
- `GA4_PROPERTY_ID`
- `SLACK_BOT_TOKEN`

## 3. .claude/.env 경로 확인

유저명이 다르면 `~/Seulki.log/.claude/.env`에서 경로 수정:

```
HFK_VAULT=/Users/{username}/Seulki.log
HFK_SKLEE=/Users/{username}/hfk-sklee
HFK_IMAGES=/Users/{username}/hfk-imweb/images
HFK_PHOTOS=/Users/{username}/hfk
```

## 4. Obsidian vault 열기

Obsidian → Open folder as vault → `~/Seulki.log` 선택

## 5. gitignore 대상 (수동 복사 필요시)

- `images/` — 상품 이미지
- `.obsidian/` — Obsidian 플러그인/테마 설정
- `HFK Slack export .../` — Slack 백업
