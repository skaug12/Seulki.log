# HFK 스킬 가이드 업데이트

`.claude/commands/` 폴더의 모든 스킬을 스캔하여 스킬 가이드를 자동 업데이트합니다.

## 사용법

```
/update-skill-guide
```

## 실행

1. `.claude/commands/` 폴더의 모든 `.md` 파일을 스캔합니다.

2. 각 스킬 파일에서 정보를 추출합니다:
   - 파일명 → 스킬명 (예: `sync-products.md` → `/sync-products`)
   - 첫 번째 `# ` 헤더 → 스킬 제목
   - `## 사용법` 섹션 → 사용법
   - 파일 내용 요약 → 주요 기능

3. 스킬 가이드 문서를 업데이트합니다:
   - 경로: `.claude/SKILL_GUIDE.md`
   - 스킬 목록 테이블 갱신
   - 각 스킬 상세 설명 갱신
   - 업데이트 이력에 오늘 날짜 추가

4. 변경 사항을 요약하여 보여줍니다:
   - 새로 추가된 스킬
   - 삭제된 스킬
   - 내용이 변경된 스킬

---

## 스킬 정보 추출 규칙

### 제목 추출
```markdown
# HFK 시즌레터 생성  →  "HFK 시즌레터 생성"
```

### 간단 설명 추출
제목 다음 첫 번째 문단 또는 파일 내용 요약

### 사용법 추출
```markdown
## 사용법

\```
/generate-letter [시즌] [레터번호] [제목]
\```
```

### 주요 기능 추출
`## 실행` 또는 `## 프로세스` 섹션의 번호 목록

---

## 출력 형식

### 스킬 목록 테이블

```markdown
| 스킬명 | 명령어 | 설명 |
|--------|--------|------|
| [시즌레터 생성](#generate-letter) | `/generate-letter` | HFK 시즌레터 HTML 자동 생성 |
| [스킬 가이드 업데이트](#update-skill-guide) | `/update-skill-guide` | 스킬 가이드 자동 갱신 |
```

### 상세 설명 섹션

```markdown
### generate-letter
**HFK 시즌레터 생성**

[간단 설명]

**사용법:**
\```
/generate-letter [시즌] [레터번호] [제목]
\```

**주요 기능:**
- [기능 1]
- [기능 2]
```

## 카테고리 분류 규칙

스킬명 키워드로 자동 분류:

| 키워드 | 카테고리 |
|--------|----------|
| `generate`, `expand` | 콘텐츠 생성 |
| `sync`, `backup`, `evaluate` | 상품/데이터 관리 |
| `schedule`, `calendar`, `daily` | 일정/캘린더 |
| `image`, `categorize` | 이미지 관리 |
| `reminder` | 미리알림 |
| `note` | NOTE 관리 |
| `skill`, `guide` | 시스템 |

---

## 주의사항

- 기존 SKILL_GUIDE.md 파일이 없으면 새로 생성합니다
- 수동으로 추가한 워크플로우 예시 섹션은 유지됩니다
- 스킬 파일이 삭제되면 가이드에서도 자동 제거됩니다
- 업데이트 이력은 파일 하단에 자동 추가됩니다
