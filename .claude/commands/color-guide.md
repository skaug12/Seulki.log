## HFK Brand Color System (Display Version / RGB)

출처: HFK Brand Identity Guideline (by ppp foundary)

브랜드의 지적 품격과 열정을 동시에 표현하는 레드 중심의 전략적 컬러 시스템입니다. 시그니처 레드를 핵심으로 하여 어두운 레드부터 밝은 코랄 톤까지 레드 계열의 정교한 변주를 통해 고급스럽고 일관된 브랜드 경험을 제공합니다. 뉴트럴 계열의 중성 톤이 균형감을 더해 과도한 자극 없이 콘텐츠 가독성을 높이며, 전체적으로 성숙하고 세련된 커뮤니티 이미지를 완성합니다.

## 사용 가이드

### 기본 조합 원칙

- **밝은 배경** (HFK White / Cream Light / Light Sand) + **HFK Red 또는 Burgundy 텍스트**
- **다크 배경** (HFK Burgundy / Charcoal / Espresso) + **HFK White 또는 Sand 텍스트** + **HFK Coral 포인트**
- **강조가 필요한 요소**: HFK Coral 또는 Salmon 사용

### 용도별 권장

| 용도 | 권장 색상 |
|------|----------|
| 웹 배경 | HFK White (#f0eae2), Cream Light (#e1dac9) |
| 본문 텍스트 | HFK Charcoal (#2d2523), Burgundy (#441a15) |
| 제목/강조 | HFK Red (#970101) |
| CTA/버튼 | HFK Coral (#ff6154) |
| 카드/섹션 구분 | HFK Sand (#ceb9a4), Light Sand (#ddcfbf) |
| 다크 모드 배경 | HFK Burgundy (#441a15), Espresso (#391200) |

---

## 주의사항

- 순수 블랙(#000000)이나 순수 화이트(#ffffff)는 사용하지 않습니다
- 레드 계열 색상끼리 인접 배치 시 명도 차이를 충분히 확보합니다
- 코랄/살몬은 포인트로만 사용하고 대면적 사용을 지양합니다
