# Seulki.log

Claude Code 커스텀 commands 저장소.

## 사용법

```bash
# 클론
git clone https://github.com/skaug12/Seulki.log.git ~/Seulki.log

# Claude Code 글로벌 commands로 symlink
ln -s ~/Seulki.log/commands ~/.claude/commands
```

## Commands 목록

`commands/` 폴더의 `.md` 파일들이 Claude Code에서 `/command-name` 형태로 사용 가능합니다.
